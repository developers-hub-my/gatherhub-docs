Save venue information in your organization's library to quickly reuse them when creating events. {% .lead %}

---

## Venue Library Benefits

- **Save time** - No need to re-enter venue details
- **Consistency** - Same information across events
- **Organization** - Central repository of locations
- **Quick selection** - Choose venue during event creation

---

## Accessing the Library

### Organization Level

1. Go to **Organization Settings**
2. Click **Venues**
3. View your venue library

### During Event Creation

1. In event creation wizard
2. At location step
3. Choose from library

---

## Adding a Venue

### Add New Venue

1. Go to **Venues**
2. Click **Add Venue**
3. Fill in details
4. Save

### Venue Information

| Field | Required | Description |
|-------|----------|-------------|
| Name | Yes | Venue name |
| Address | Yes | Full street address |
| City | Yes | City |
| State | No | State/Province |
| Postal Code | No | ZIP/Postal code |
| Country | Yes | Country |
| Capacity | No | Maximum attendees |
| Description | No | Notes about venue |

### Additional Details

| Field | Description |
|-------|-------------|
| Map coordinates | Latitude/longitude |
| Parking info | Parking availability |
| Accessibility | Accessibility features |
| Contact | Venue contact person |
| Website | Venue website |

---

## Managing Venues

### Edit Venue

1. Find venue in library
2. Click **Edit**
3. Update information
4. Save changes

### Delete Venue

1. Find venue
2. Click **Delete**
3. Confirm deletion

{% callout type="warning" title="Deleting venues" %}
Deleting a venue from the library doesn't affect events already using it. Those events keep the venue information.
{% /callout %}

---

## Using Venues in Events

### Select from Library

1. Create or edit event
2. Go to Location settings
3. Click **Choose from Library**
4. Select venue
5. All details are auto-filled

### Override Details

After selecting:

- Edit specific fields for this event
- Original library entry unchanged
- Event has its own copy

---

## Venue Best Practices

### Information to Include

- Complete address
- Parking instructions
- Public transport access
- Entry point details
- Accessibility information
- Contact for day-of issues

### Organization

- Use consistent naming
- Keep information current
- Include useful notes
- Add photos if possible
