The real-time dashboard gives you instant visibility into attendance as participants check in to your event. {% .lead %}

---

## Dashboard Overview

The check-in dashboard displays:

- Live check-in counter
- Recent check-ins
- Attendance statistics
- Session breakdown
- Check-in activity feed

---

## Accessing the Dashboard

### From Event Dashboard

1. Go to your event
2. Click **Check-In** in the sidebar
3. Dashboard loads automatically

### Display Mode

For venue display:

1. Open dashboard
2. Click **Display Mode**
3. Full-screen stats view
4. Auto-refreshes continuously

---

## Dashboard Elements

### Check-In Counter

Large display showing:

| Metric | Description |
|--------|-------------|
| Checked In | Number checked in |

## Troubleshooting

### Dashboard not updating

1. Check internet connection
2. Verify browser is active
3. Click manual refresh
4. Reload the page

### Incorrect counts

1. Allow time to sync
2. Check offline check-ins
3. Verify duplicate entries
4. Review recent activity

### Slow performance

1. Close unused tabs
2. Use simpler view
3. Reduce refresh rate
4. Check device memory
