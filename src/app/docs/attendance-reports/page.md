## Attendance Overview

### Key Metrics

| Metric | Description |
|--------|-------------|
| Total Registered | Confirmed registrations |
| Checked In | Participants who arrived |
| Check-In Rate | Percentage attended |
| No-Shows | Registered but didn't attend |

## Report Types

### Event Attendance

Overall event check-in data:

- Total check-ins
- Check-in timeline
- Peak arrival times
- Attendance by ticket type

### Session Attendance

Per-session tracking:

- Attendance by session
- Capacity utilization
- Popular sessions
- Session comparison

### Activity Attendance

Per-activity participation:

- Activity completion
- Participation rates
- Activity comparison

---

## Generating Reports

### Create Attendance Report

1. Go to **Reports**
2. Select **Attendance**
3. Choose parameters:
   - Date range
   - Session/activity filter
   - Ticket type filter
4. Generate report

### Report View

| Column | Description |
|--------|-------------|
| Participant | Name |
| Email | Contact email |
| Ticket | Ticket type |
| Check-In Time | When checked in |
| Sessions | Sessions attended |
| Activities | Activities completed |
| Attendance % | Overall percentage |
