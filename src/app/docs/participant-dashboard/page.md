The participant portal gives registered attendees a personal dashboard to manage their event registrations, tickets, and certificates. {% .lead %}

---

## Accessing the Portal

### How to Access

Participants can access their portal:

1. **From confirmation email** - Click portal link
2. **Direct URL** - gatherhub.app/me
3. **From event page** - Click "My Registrations"

### Login Options

- Email and password
- Social login (if enabled)
- Magic link (passwordless)

---

## Dashboard Overview

### What Participants See

| Section | Content |
|---------|---------|
| My Events | All registered events |
| Upcoming | Future events |
| Past | Completed events |
| Certificates | Available certificates |

### Event Cards

Each event shows:

- Event name and date
- Registration status
- Check-in status
- Quick actions

---

## Event Categories

### Upcoming Events

Events in the future:

- Event details
- Countdown to event
- View ticket option
- Add to calendar

### Ongoing Events

Currently active events:

- Check-in status
- Session schedule
- Live updates

### Past Events

Completed events:

- Attendance summary
- Certificate availability
- Event archive

---

## Dashboard Features

### Quick Actions

| Action | Description |
|--------|-------------|
| View Ticket | Display ticket with QR code |
| View Event | Go to event details |
| Download Certificate | Get certificate if available |
| Contact Organizer | Send message |

### Notifications

Participants receive:

- Registration confirmation
- Event reminders
- Certificate availability
- Important updates

## Best Practices for Organizers

### Portal Experience

Ensure participants can:

- Easily find their tickets
- Access event information
- Download certificates
- Contact support

### Communication

- Include portal links in emails
- Remind about certificate access
- Provide help documentation
