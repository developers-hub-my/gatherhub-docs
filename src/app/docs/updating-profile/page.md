## Accessing Profile Settings

### How to Access

1. Log in to participant portal
2. Click your name or avatar
3. Select **Profile** or **Settings**

---

## Personal Information

### Editable Fields

| Field | Description |
|-------|-------------|
| Full Name | Your display name |
| Email | Primary contact email |
| Phone | Contact phone number |
| Profile Photo | Your avatar/photo |

### Updating Information

1. Go to Profile
2. Click **Edit**
3. Update fields
4. Save changes

---

## Profile Photo

### Adding/Changing Photo

1. Go to Profile
2. Click on photo or **Change Photo**
3. Upload new image
4. Adjust cropping
5. Save

## Account Security

### Password

Change your password:

1. Go to **Settings** > **Security**
2. Click **Change Password**
3. Enter current password
4. Enter new password
5. Confirm and save

### Two-Factor Authentication

If available:

1. Go to **Security**
2. Enable **Two-Factor Authentication**
3. Scan QR with authenticator app
4. Enter verification code
5. Save backup codes

## Notification Preferences

### Email Notifications

Control which emails you receive:

| Notification | Description |
|--------------|-------------|
| Event reminders | Before events |
| Registration updates | Status changes |
| Certificate available | When ready |
| Event announcements | Organizer updates |

### Managing Preferences

1. Go to **Settings** > **Notifications**
2. Toggle notifications on/off
3. Save preferences

## Account Actions

### Manage Account

| Action | Description |
|--------|-------------|
| Download data | Export your information |
| Deactivate account | Temporarily disable |
| Delete account | Permanently remove |

### Data Export

Download your data:

1. Go to **Settings** > **Privacy**
2. Click **Download My Data**
3. Receive file with your information

## Troubleshooting

### Can't update profile

1. Check required fields
2. Verify email format
3. Refresh page
4. Contact support

### Photo not uploading

1. Check file size
2. Try different format
3. Clear browser cache
4. Try different browser

### Password issues

1. Use forgot password
2. Check password requirements
3. Contact support

## Best Practices

### Keep Information Current

- Update after changes
- Use accurate email
- Keep phone current

### Security

- Strong, unique password
- Enable two-factor if available
- Review security regularly
