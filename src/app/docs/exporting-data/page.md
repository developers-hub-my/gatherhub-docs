## Export Options

### Available Exports

| Data Type | Description |
|-----------|-------------|
| Participants | Registration and attendee data |
| Orders | Transaction records |
| Attendance | Check-in records |
| Certificates | Certificate issuance |
| Custom Fields | Custom field responses |

### Export Formats

| Format | Best For |
|--------|----------|
| CSV | Universal compatibility |
| Excel (.xlsx) | Spreadsheet analysis |
| PDF | Printable reports |

---

## Exporting Participants

### How to Export

1. Go to **Participants**
2. Apply filters (optional)
3. Click **Export**
4. Choose format
5. Download

### Available Fields

| Field | Description |
|-------|-------------|
| Name | Full name |
| Email | Email address |
| Phone | Phone number |
| Ticket Type | Registration type |
| Status | Registration status |
| Check-In | Attendance status |
| Custom Fields | All custom responses |

### Filtering Before Export

Export specific segments:

- By registration status
- By ticket type
- By check-in status
- By date range

---

## Exporting Orders

### Transaction Export

1. Go to **Orders**
2. Filter if needed
3. Click **Export**
4. Choose format

### Order Fields

| Field | Description |
|-------|-------------|
| Order Number | Unique identifier |
| Date | Order date |
| Participant | Buyer name |
| Email | Buyer email |
| Tickets | Quantity and type |
| Amount | Total paid |
| Payment Method | How they paid |
| Status | Order status |

## Exporting Attendance

### Check-In Export

1. Go to **Reports** > **Attendance**
2. Set date range
3. Export

### Attendance Fields

| Field | Description |
|-------|-------------|
| Participant | Name |
| Ticket | Ticket type |
| Event Check-In | Time checked in |
| Sessions | Sessions attended |
| Activities | Activities completed |
| Percentage | Attendance % |

## Custom Exports

### Selecting Fields

1. Click **Export**
2. Choose **Custom Export**
3. Select fields to include
4. Order fields as needed
5. Export

### Saving Export Templates

Save commonly used configurations:

1. Create custom export
2. Click **Save as Template**
3. Name the template
4. Reuse for future exports

---

## Scheduled Exports

### Automatic Exports

Set up recurring exports:

1. Go to **Export Settings**
2. Click **Schedule Export**
3. Configure:
   - Data type
   - Frequency
   - Format
   - Delivery (email or storage)

### Use Cases

- Daily registration reports
- Weekly attendance summaries
- Post-event data delivery

## Data Privacy

### Sensitive Data

Consider privacy when exporting:

- Personal information
- Payment details
- Contact information

### Best Practices

- Export only needed data
- Secure exported files
- Delete when no longer needed
- Follow data protection regulations

## Bulk Operations

### Large Exports

For events with many participants:

- Exports may take time
- Download link sent when ready
- Files compressed for size

### Export Limits

| Size | Behavior |
|------|----------|
| < 1,000 rows | Immediate download |
| 1,000-10,000 | Background processing |
| > 10,000 | Split into multiple files |

## Troubleshooting

### Export not downloading

1. Check popup blocker
2. Try different browser
3. Clear cache
4. Contact support

### Missing data

1. Check applied filters
2. Verify field selection
3. Refresh and retry
4. Check data exists

### Formatting issues

1. Open in correct application
2. Check CSV delimiter
3. Try different format
4. Check character encoding

## Best Practices

### Regular Exports

- Export after major milestones
- Keep backups of important data
- Export before event ends

### Organization

- Use consistent file naming
- Date your exports
- Store securely
- Document what was exported
