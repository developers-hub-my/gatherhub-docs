## Sponsorship Tiers

### Common Tiers

| Tier | Description |
|------|-------------|
| Platinum | Highest level sponsor |
| Gold | Premium sponsor |
| Silver | Standard sponsor |
| Bronze | Supporting sponsor |
| Partner | Non-financial partners |

### Creating Tiers

1. Go to **Organization Settings** > **Sponsor Tiers**
2. Click **Add Tier**
3. Configure:

| Field | Description |
|-------|-------------|
| Name | Tier name |
| Order | Display order |
| Benefits | What sponsors receive |
| Logo size | Display size |

---

## Adding Sponsors

### Add to Event

1. Go to event dashboard
2. Click **Sponsors**
3. Click **Add Sponsor**
4. Fill in details

### Sponsor Information

| Field | Description |
|-------|-------------|
| Company name | Sponsor's name |
| Logo | Company logo |
| Tier | Sponsorship level |
| Website | Sponsor's website |
| Description | About the sponsor |

### Logo Requirements

| Spec | Recommendation |
|------|----------------|
| Format | PNG, JPG, SVG |
| Size | 400Ã—200 px minimum |

## Sponsor Display

### On Event Page

Configure sponsor visibility:

| Setting | Description |
|---------|-------------|
| Show sponsors | Enable/disable display |
| Logo size by tier | Larger for higher tiers |
| Order | By tier, then alphabetical |
| Link to website | Click logo to visit site |

### Display Options

- Logo only
- Logo with name
- Logo with description
- Grouped by tier

---

## Managing Sponsors

### Edit Sponsor

1. Go to **Sponsors**
2. Click on sponsor
3. Update details
4. Save

### Reorder Sponsors

- Drag to reorder within tier
- Higher tiers display first

### Remove Sponsor

1. Find sponsor
2. Click **Remove**
3. Confirm removal

---

## Sponsor Library

### Save to Library

Reuse sponsors across events:

1. After adding sponsor
2. Click **Save to Library**
3. Available for future events

### Use from Library

1. Click **Add from Library**
2. Select sponsor
3. Assign tier for this event

### Communication
