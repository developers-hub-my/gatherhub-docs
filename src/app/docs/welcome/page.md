## What is GatherHub?

GatherHub is a comprehensive event management platform designed for organizations that need to:

- **Manage registrations** for trainings, workshops, seminars, and conferences
- **Process payments** securely through Malaysian payment gateways
- **Track attendance** at multiple levels (event, session, activity)
- **Issue certificates** with verifiable credentials
- **Communicate** with participants before, during, and after events

{% callout title="Who is GatherHub for?" %}
GatherHub is built for event organizers in Malaysia who run recurring programs like corporate trainings, community workshops, professional seminars, and educational conferences.
{% /callout %}

## Key Features

### Event Creation and Management

Create events with a guided wizard that walks you through:

1. Basic information (title, description, dates)
2. Location setup (physical, virtual, or hybrid)
3. Ticketing and pricing configuration
4. Registration form customization

### Multi-Level Event Structure

Organize complex events with:

- **Sessions** - Time-blocked segments (e.g., morning session, afternoon session)
- **Activities** - Specific workshops or presentations within sessions
- **Custom fields** - Collect additional data from participants

### Ticketing and Payments

- Multiple ticket types with different prices and availability windows
- Early bird pricing and discount codes
- Payment processing via FPX, DuitNow QR, and manual bank transfer
- Automatic waitlist handling when tickets sell out

### Check-In System

- QR code scanning for fast, contactless check-in
- Multi-level tracking (event, session, activity)
- Real-time attendance dashboard
- Mobile-friendly interface for check-in stations

### Certificate Generation

- Customizable certificate templates with your branding
- Eligibility rules based on attendance requirements
- QR code verification for authenticity
- Batch generation for all qualifying participants

### Team Collaboration

- Invite crew members with specific roles
- Role-based permissions (organizer, coordinator, staff, volunteer, speaker)
- Delegate tasks while maintaining control

## How GatherHub Works

### For Event Organizers

1. **Create** your organization and invite team members
2. **Build** events using the creation wizard
3. **Configure** tickets, registration forms, and payment settings
4. **Publish** and share your event page
5. **Manage** registrations and communicate with participants
6. **Run** check-in on event day
7. **Generate** certificates and export reports

### For Participants

1. **Discover** events on your organization's event page
2. **Register** and complete payment
3. **Receive** confirmation email with ticket and QR code
4. **Attend** the event and check in with QR code
5. **Access** the participant portal to view tickets and certificates
