## Speaker Management

### Features

- Central speaker directory
- Reusable across events
- Profile with bio and photo
- Session assignments
- Speaker portal access

---

## Adding Speakers

### Add to Library

1. Go to **Organization Settings** > **Speakers**
2. Click **Add Speaker**
3. Fill in profile
4. Save

### Speaker Profile Fields

| Field | Description |
|-------|-------------|
| Name | Full name |
| Email | Contact email |
| Photo | Profile picture |
| Title | Job title |
| Organization | Company/organization |
| Bio | Short biography |
| Expertise | Areas of expertise |
| Social links | LinkedIn, Twitter, etc. |

---

## Event Speaker Assignment

### Add Speaker to Event

1. Go to event dashboard
2. Click **Speakers**
3. Click **Add Speaker**
4. Choose from library or create new
5. Configure for this event

### Event-Specific Settings

| Setting | Description |
|---------|-------------|
| Display on page | Show on public event page |
| Session assignments | Which sessions they present |
| Speaker role | Keynote, panelist, workshop leader |

---

## Session Assignment

### Assign to Sessions

1. Go to **Sessions**
2. Edit session or activity
3. Select speaker(s)
4. Save

### Multiple Speakers

Sessions can have multiple speakers:

- Panel discussions
- Co-presented workshops
- Team presentations

---

## Speaker Display

### On Event Page

Speaker information shown:

- Name and photo
- Title and organization
- Bio
- Sessions presenting
- Social links

### Customizing Display

Control what's visible:

- Show/hide specific speakers
- Order of appearance
- Which fields to display

---

## Speaker Portal

### Speaker Access

Speakers can:

- View their sessions
- See attendee count
- Access session materials
- Update their bio

### Inviting Speakers

1. Add speaker to event
2. Click **Invite to Portal**
3. They receive access link
4. Can log in to view

---

## Best Practices

### Profile Quality

- Professional photo
- Concise, engaging bio
- Relevant credentials
- Updated information

### Event Integration

- Assign speakers early
- Confirm session details
- Provide speaker guidelines
- Communicate logistics
