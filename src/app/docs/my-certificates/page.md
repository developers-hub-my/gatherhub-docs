## Accessing Certificates

### Where to Find

1. Log in to participant portal
2. Click **My Certificates**
3. View all available certificates

### Or From Event

1. Go to specific event
2. Click **Certificates** section
3. View certificates for that event

---

## Certificate List

### What You See

| Column | Description |
|--------|-------------|
| Event | Event name |
| Certificate | Certificate title |
| Date Issued | When generated |
| Status | Active or Revoked |
| Actions | Download options |

---

## Downloading Certificates

### Download Options

| Format | Description |
|--------|-------------|
| PDF | Standard document format |
| PNG | Image format |

### How to Download

1. Find certificate
2. Click **Download**
3. Choose format
4. File downloads

---

## Certificate Details

### Viewing Details

Click on certificate to see:

- Full certificate preview
- Participant name
- Event information
- Certificate number
- Issue date
- Verification QR code

### Certificate Number

Unique identifier format:
```
CERT-EVENT-2024-00001
```

Used for verification.

---

## Certificate Verification

### Verify Your Certificate

1. Scan QR code on certificate
2. Or visit verification URL
3. Enter certificate number
4. See verification result

### What Verification Shows

- Certificate validity
- Your name
- Event details
- Issue date

### Sharing Verification

Provide to employers/others:

- Certificate number
- Verification URL
- QR code on certificate

---

## Certificate Eligibility

### Why You Got a Certificate

Certificates are issued when you:

- Attended required sessions
- Met attendance percentage
- Completed required activities
- Qualified per event rules

### No Certificate Available?

Possible reasons:

| Reason | What to Do |
|--------|------------|
| Not yet issued | Wait for organizer |
| Didn't meet requirements | Check with organizer |
| Technical issue | Contact organizer |

---

## Multiple Certificates

### Different Certificate Types

An event may have:

- Attendance certificate
- Completion certificate
- Achievement certificate
- Special recognition

Each appears separately in your list.

## Printing Certificates

### Print Quality Tips

- Use high-quality paper
- Print at actual size
- Check printer settings
- Use color printing

### Recommended Paper

- Standard A4 or Letter
- Certificate paper for formal look
- Check certificate orientation

---

## Troubleshooting

### Certificate not showing

1. Check event has ended
2. Verify you met requirements
3. Wait for organizer to generate
4. Contact organizer

### Can't download

1. Try different browser
2. Check popup blocker
3. Try different format
4. Contact support

### Wrong information

1. Contact event organizer
2. Request correction
3. New certificate generated
