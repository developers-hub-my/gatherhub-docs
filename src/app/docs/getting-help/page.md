## When Contacting Support

### Include This Information

Help us help you faster by providing:

| Information | Why It Helps |
|-------------|--------------|
| Your email | Identify your account |
| Event name | Locate the event |
| Description | Understand the issue |
| Screenshots | See what you're seeing |
| Steps taken | Know what you've tried |
| Browser/device | Check compatibility |

### Example Support Request

```
Subject: Cannot download certificate - [Event Name]

Hi,

I attended [Event Name] on [Date] but cannot download my certificate.

What I tried:
- Clicked Download button
- Tried different browsers (Chrome, Safari)
- Cleared cache

Error message: "Download failed"

My email: [your email]
Event: [Event Name]
Browser: Chrome 120 on macOS

[Attach screenshot]

Thank you!
```

## Emergency Support

### Critical Event Issues

If you have a critical issue during a live event:

1. Email support@gatherhub.app with "URGENT" in subject
2. Include event name and issue
3. Provide contact phone number
4. We'll prioritize your request
