---

## Viewing Registrations

### Registration List

Access all registrations:

1. Log in to participant portal
2. Click **My Registrations**
3. View all registered events

### Registration Details

Each registration shows:

| Info | Description |
|------|-------------|
| Event Name | Event you're registered for |
| Date | When the event takes place |
| Ticket Type | Your ticket |
| Status | Confirmed, Pending, etc. |
| Check-In | Whether you've checked in |

---

## Ticket Information

### Viewing Your Ticket

1. Click on a registration
2. Select **View Ticket**
3. See ticket details and QR code

### Ticket Contents

| Element | Description |
|---------|-------------|
| QR Code | For check-in scanning |
| Ticket Number | Unique identifier |
| Event Details | Name, date, location |
| Your Name | Registered name |

### Using Your Ticket

- Show QR code on mobile device
- Print ticket if preferred
- Have ticket number as backup

---

## Session Registration

### If Sessions Available

For events with sessions:

1. View registration details
2. See **My Sessions** section
3. View registered sessions

### Session Information

| Info | Description |
|------|-------------|
| Session Name | Title of session |
| Time | Start and end time |
| Location | Room or area |
| Status | Registered, Attended |

---

## Modifying Registrations

### What You Can Change

Depending on organizer settings:

| Action | Availability |
|--------|--------------|
| Update personal info | Usually allowed |
| Change sessions | If enabled |
| Cancel registration | If allowed |
| Transfer ticket | If enabled |

### How to Modify

1. Open registration
2. Click **Edit** (if available)
3. Make changes
4. Save

---

## Cancelling Registration

### If Cancellation Allowed

1. Open registration
2. Click **Cancel Registration**
3. Confirm cancellation
4. Review refund policy

### Cancellation Effects

- Registration cancelled
- Spot may be released
- Refund per event policy
- Confirmation email sent

---

## Registration Status

### Status Types

| Status | Meaning |
|--------|---------|
| Confirmed | Registration complete |
| Pending Payment | Awaiting payment |
| Pending Approval | Awaiting organizer |
| Cancelled | Registration cancelled |

### Status Actions

| Status | You Can |
|--------|---------|
| Confirmed | View ticket, check in |
| Pending Payment | Complete payment |
| Pending Approval | Wait for decision |
| Cancelled | Re-register if open |

## Event Day

### Before the Event

- Verify registration status
- Download/save ticket
- Review event details
- Note check-in location

### During the Event

- Show QR code for check-in
- Check session schedule
- Track your attendance

## Troubleshooting

### Can't find registration

- Check email for confirmation
- Verify you're logged in correctly
- Contact organizer

### Wrong information

- Edit if allowed
- Contact organizer for changes
- Provide correct details

### Registration not confirmed

- Check payment status
- Complete pending payment
- Contact organizer
